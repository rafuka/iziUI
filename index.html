Aside menu (components)

iziUI is a easy-to-use UI framework. The use of flex-box containers instead of columns and rows makes it a simple framework for a quick responsive solution!

Layout

The main layout components are containers, which come in two flavors: container-row and container-col, and correspond to the flex-direction: column and flex-direction: row properties.

This makes it easy to create responive layouts in seconds! Plus it is highly customizable using the flexbox properties.
