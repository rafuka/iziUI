iziUI is an easy-to-use UI framework based on CSS flexbox.

Layout

The use of flex-box containers instead of columns and rows makes it a simple framework for a quick responsive solution!

The main layout components are containers, which come in two flavors — row containers and column containers — and correspond to the flex-direction: column and flex-direction: row properties.

iziUI allows you to create responsive layouts in seconds! Plus it is highly customizable using the flexbox properties.
